# Sistema de Oficina Mecânica

## Sobre o projeto
Esse projeto foi feito para a disciplina de Engenharia de Requisitos.

A ideia é criar um sistema para ajudar oficinas mecânicas a organizar ordens de serviço e acompanhar os serviços realizados.

## O que o sistema faz
- Cadastrar clientes
- Registrar ordens de serviço
- Atualizar o status da ordem
- Ver histórico de serviços

## Estrutura
O projeto está organizado assim:

- docs: documentação e requisitos
- src: onde ficará o código
- tests: testes do sistema
- infra: arquivos de configuração
