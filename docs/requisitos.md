# Requisitos do Sistema

## Requisitos Funcionais
- RF01: Cadastrar cliente
- RF02: Registrar ordem de serviço
- RF03: Atualizar status da ordem
- RF04: Listar ordens de serviço
- RF05: Consultar histórico de serviços
- RF06: Associar veículo ao cliente

## Requisitos Não Funcionais
- Interface simples e intuitiva
- Sistema deve responder rapidamente
- Dados devem ser consistentes
- Sistema deve permitir fácil manutenção
